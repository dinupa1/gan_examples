# Generative Adversarial Network (GAN)

The original data file is located at [https://github.com/JeffersonLab/jlab_datascience_data/blob/main/eICU_age.npy](https://github.com/JeffersonLab/jlab_datascience_data/blob/main/eICU_age.npy).

We implemented four different methods to generate the age distribution of critical care patients across intensive care units in the United States:

1. [Vanilla GAN](https://arxiv.org/abs/1406.2661)
2. [Instance Noise Method](https://arxiv.org/abs/1610.04490)
3. [DCTR Reweighting](https://arxiv.org/abs/2009.03796)
4. [VAE (Variational Autoencoder)](https://arxiv.org/abs/1312.6114)
